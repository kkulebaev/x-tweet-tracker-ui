# x-tweet-tracker-ui

Web admin UI for x-tweet-tracker.

## Environment variables

This UI expects the same *logical* variables as the bot, but in Vite form:

- `VITE_API_BASE_URL` (from `API_BASE_URL`)
- `VITE_API_TOKEN` (from `API_TOKEN`)

## Local dev

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
```
